Some Ruby History
  • Ruby was developed in 1993 by Yukihiro “Matz” Matsumoto
  • His intention was to create “a scripting language more powerful than Perl, and more object-oriented than Python”
  • Bonus: Unlike Perl, you code doesn’t look like a giant regular expression when you’ve finished

Defining a Class
  • Simple

  attr_reader :make, :model, :year

  def initialize(make, model, year)
    @make, @model, @year = make, model, year
  end

  def drive(distance)
    puts "Drove a #{@year} #{@make} #{@model} #{distance}"
  end

end

Enumerable
  • select, reject, detect
    • files.select { |f| File.stat(f) > 70_000 }  # ['zsh.tar']

files.detect { |f| File.stat(f) > 70_000 }  # 'zsh.tar'

files.reject { |f| File.stat(f) > 70_000 }  # ['index.xml']
  • inject
    • files.inject({}) do |manifest, filename|
  manifest[filename] = File.stat(filename).size
  manifest
end

Mixins

  KM_PER_MILE = 1.609

  def mi_to_km(miles)
    miles * KM_PER_MILE
  end

  def km_to_mi(kilometers)
    kilometers / KM_PER_MILE
  end

end
